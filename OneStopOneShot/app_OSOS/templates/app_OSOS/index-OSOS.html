Welcome to One-Stop One-Shot!

OSOS is a project that incorporates Python, Django, Javascript, Vue, Html, and Css. Its purpose is to demonstrate my proficiency with these languages/frameworks by incorporating multiple apps into a single project that allows someone to quickly build a One-shot session.

One-shots are tabletop role playing game scenarios that are concluded in a single session. When last minute emergencies or scheduling conflicts prevent a group of tabletop gamers from playing their usual campaign, OSOS will allow them to quickly build a replacement scenario to play instead.

Using OSOS is simple. Just follow these steps.

1. Signup or Login
2. Go to Profile
3. Go to new Portfolio
4. Create a villain/antagonist using the Villain Generator
5. Select a battlemap or battlemaps with the Map Generator
6. Create combat encounters using the Encounter Builder
7. Enjoy your completed portfolio!
